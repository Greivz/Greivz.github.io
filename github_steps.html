Github Page Steps

Bagaimana sih cara buatnya ??

1

Pastikan memiliki akun github

Jika sudah memiliki akun, Login ke akun Github kalian.

2

buat repositori

Untuk membuat repositori, kalian bisa klik Repository → New → Masukkan nama repository sesuai keinginan, di sini aku menggunakan username.github.io → Klik Create repository

3

Buka folder pada local directory

Setelah masuk ke dalam folder yang berisi file-file kalian, klik kanan → Pilih Git Bash Here

4

Inisiasi repository

Lakukan inisisasi repositori dengan perintah 'git init'

5

Masukkan ke stagging area

Untuk memasukkan file-file ke stagging area, gunakan perintah 'git add .' titik di sini artinya semua file.

6

Lakukan commit

Untuk melakukan commit, gunakan perintah 'git commit -m"commit pertama"'. "commit pertama" dapat diubah menjadi kata apapun yang dapat memudahkan kalian.

7

Lakukan remote repositori

Untuk melakukan remote repositori, gunakan perintah 'git remote add origin https://github.com/Greivz/Greivz. github.io.git'. link dapat diubah sesuai dengan nama repositori kalian

8

Push file ke remote (Github) repository

Untuk memasukkan file ke remote, gunakan perintah 'git push origin master'. Lalu akan muncul tampilan untuk memasukkan token.

9

Salin token dan Sign In

Untuk mendapatkan token dapat kembali ke halaman Github, lalu klik setting dan pilih Developer settings → Personal access tokens → masukkan Note → klik Generate token → Masukkan password Github → Salin token → klik Sign In → File sudah berhasil di-push → cek di repository Github.

Untuk lebih jelas, silakan klik tombol di bawah ini untuk melihat laporannya.
